MANUAL MUST - PWA

1. Para testar no computador, abra index.html.
2. Para funcionar como PWA instalável, publique esta pasta em HTTPS, por exemplo Vercel, Netlify ou GitHub Pages.
3. Android/Chrome: abrir o link publicado e tocar em Instalar app ou Adicionar à tela inicial.
4. iPhone/iPad/Safari: abrir o link publicado, tocar em Compartilhar e selecionar Adicionar à Tela de Início.
5. O service worker permite funcionamento offline depois do primeiro acesso online.
